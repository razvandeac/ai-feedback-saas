What's Happening?

When you click a button above, the widget SDK sends feedback to the PulseAI API. The feedback is stored in the database and can be viewed in your dashboard.

SDK Methods

  • PulseAI.init(config) - Initialize the SDK with your project ID
  • PulseAI.capture(payload) - Capture and send feedback
  • PulseAI.isInitialized() - Check if SDK is initialized

Events

The SDK dispatches custom events you can listen to:

  • pulseai:captured - Fired when feedback is successfully captured
  • pulseai:error - Fired when an error occurs
